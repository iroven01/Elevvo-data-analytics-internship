# Elevvo Pathways Data Analytics Internship Tasks

This repository contains the tasks I completed during the Elovvo Pathways Data Analytics Internship Program. The internship focused on building practical skills in data cleaning, exploratory data analysis (EDA), visualization, and business insights generation using industry tools.

# Tools & Technologies
* Microsoft Excel
* Power BI
* Python (Pandas, Matplotlib, Seaborn)
* Jupyter Notebook

# Tasks Completed

Task 1 – Data Cleaning (Excel)
Performed data cleaning and preparation using Microsoft Excel. This included handling missing values, correcting inconsistent data formats, and preparing the dataset for analysis.

Task 2 – Exploratory Data Analysis (Python)
Used Python libraries such as Pandas and Matplotlib to explore the dataset, identify patterns, and analyze relationships between variables.

Task 8 – Data Visualization (Power BI)

Task 4 – Data Analysis with Python
